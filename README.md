## ðŸš€ Running Locally

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/leandromlc/leandromlc-portfolio.git](https://github.com/leandromlc/leandromlc-portfolio.git)
    ```

2.  **Enter the project folder**
    *(Note: The application code is inside the `/portfolio` directory)*
    ```bash
    cd leandromlc-portfolio/portfolio
    ```

3.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

4.  **Start the server**
    ```bash
    npm run dev
    ```

The project will be available at `http://localhost:3000`.
