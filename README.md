## üöÄ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/en/) (v18 or higher) installed on your machine.

### Installation

1.  **Clone the repo**
    ```bash
    git clone https://github.com/leandromlc/leandromlc-portfolio.git
    ```

2.  **Navigate to the project directory**
    *(Note: The application code is inside the `/portfolio` folder)*
    ```bash
    cd leandromlc-portfolio/portfolio
    ```

3.  **Install NPM packages**
    ```bash
    npm install
    ```

4.  **Run the development server**
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
